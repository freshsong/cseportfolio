리액트로 개인 포트폴리오 블로그 만들기

🖇 Development URL


https://github.com/freshsong/reblog

🚩 Summary


요즘 기업체에서 리액트를 많이 사용한다고 하기에, 어떤 프로그램이길래 많이쓰는지 궁금해서 공부하게 되었습니다.

개인 포트폴리오를 한참 만들어야 했을땐데, 간단한 소개와 구역을 Vscode에서 했던 방식대로 그대로 구현한다 생각하고 만들었습니다.

vscode로 리액트를 만졌는데, 앞서 HTML로 간단하게 코드를 구현했다면 좀 색다른 방식이었습니다.

블로그는 최대한 깔끔하게 구성하고 싶었습니다.


🧐 주요기능

  1. 프론트만 구현
  2. abot / story / project / contact 클릭시 화면 아래로 내려가며 각 구획별 내용 확인

🔍 Meaning


아무것도 모른 상태로 처음 리액트를 봤을때, 실제로 파일을 생성하고 화면에 띄웠을 때 파란색 로고가 보이면서 오......? 이건뭐지.....? 하면서 두근댔습니다.

이 프로그램을 만지면서 리액트로 HTML에서 했던 방법과 똑같이 블로그가 만들어진다는 점이 놀라웠습니다. 코딩방식도 비슷한듯 완전 달랐고, npm으로 필요한 모든 것들을 install 시켜서 import해준 뒤 끌어다 쓰는 방법이 독특했습니다.

이 프로그램은 혹시 HTML로 고통받은 사람이 만든 하나의 걸작이 아닌가 싶을 정도로 신기했고, 간단한 페이지를 만들었지만 보여지는것과 달리 코드는 꽤 길었고 점하나 잘못 찍으면 바로 에러로 화면이 보이지 않아 에러 찾는데 즐거움도 있었습니다.


🔨 Technology Stack

  1. Frontend : Javscript, CSS, HTML, Bootstrap, react
  2. Backend : nodejs
