Java(JSP/Servlet)로 게시판 만들기

🖇 Development URL


https://github.com/freshsong/JSPboard

🚩 Summary


자바로 만드는 간단한 게시판 사이트 입니다. 웹페이지 제작시 가장 기본적이고 보편적으로 공부하는 내용이고 저또한 자바로 실제로 게시판 구현이 가능한가? 궁금점이 들어서 개발했습니다.

매번 프론트 관련된 프로그램, 코딩만 짜다가 MYSQL이라는 백엔드 DB 프로그램도 접해봤고, select * from... 낯선 구문에 조금 당황할때도 있었지만, 여지껏 보여지는 코딩에 집중했다면 실제로 내가 입력한 자료들이 DB에 저장이 되고 그 저장된 정보를 실제로 확인이 가능하다는 점에서 흥미로웠습니다.


🧐 주요기능

  1. 회원가입/로그인/로그아웃
  2. 회원만 글쓰기 가능
  3. 비회원은 회원가입 후 글쓰기 가능
  4. 글수정은 글작성시 넣었던 정보로 비밀번호 체크 후 수정가능

🔍 Meaning

자바를 처음 접하기 전에는 자바와 자바스크립트가 동일한 기능을 구현하는줄 알았습니다. 하지만, 전혀 다른 프로그램이었습니다.

게시판을 만든 이유는 자바를 배우는 사람이라면 한번쯤 궁금해할법한 "저 기능은 어떻게 만들어지는거지?" 라는 생각을 할 것입니다.

저는 매번 프론트 관련 코딩만 했다가, 게시판 코딩을 진행하던 중 "와 이건 어떻게 구현되는걸까?" 라는 의문점에 시작하게 되었습니다.

내가 게시판을 만들어 배포한다면, 내가 알리고싶은 정보들을 기재해서 알려줄수도 있고 그 아래에 댓글을 작성해서 나에게 내가 알지 못했던 정보들을 알려줄 수 있을것 같아서 시작하게 되었습니다.

JSP / Servlet / Dao / Dto 의 기능을 보다 더 정확하게 습득할 수 있었습니다.


🔨 Technology Stack

  1. Frontend : Javscript, CSS, HTML, Bootstrap, Jquery
  2. Backend : Java(11version), MySQL
