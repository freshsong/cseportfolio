🚩 Summary


공부하던 중 문득 웨딩사진을 보고 웨딩홀 관련 홈페이지 메인을 만들까? 해서 만들었습니다.

메인만 보면 행복해보이게끔 만들어 적은 기능으로 화려하게 보이고 싶었습니다.

최대한 깔끔하고 은은한 화려함으로 색다른 홈페이지를 만들고 싶었고 조금 만족스러웠습니다.


🧐 주요기능

  1. 회원가입 / 로그인 / 로그아웃
  2. 글등록, 댓글등록
  3. 내가 올린 관심사진, 관심유튜브 자료 확인

🔍 Meaning


공부를 하면서 결혼이야기가 나왔는데, 결혼? 저랑은 관련이 없지만 웨딩 사진을 보고 오~ 홈페이지 만들면 좋겠다! 생각해서 딱 들어와서 메인 상단을 봤을때 나도 이렇게 결혼해보고싶다 라는 생각이 들도록 만들고 싶었습니다.

깔끔한듯 아닌듯 여러 사진을 넣어서 그동안 공부한 내용들을 최대한 접목시켜보려고 했고, 깔끔하게 잘 제작된 것 같아 뿌듯했습니다.

홈페이지를 만들면서 느낀점은, 아직 홈페이지에 대해 구성을 시작할 때 뚜렷한 각 구역별 구성 UI 이런게 뚜렷하게 생각나지 않아 여러번 왔다갔다 하면서 작업을 하곤했는데, 많이 수정하고 다듬었지만 그래도 깔끔하게 잘 완성되어 기분이 좋았습니다.
