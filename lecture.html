대학교 수강신청 홈페이지 만들기

🖇 Development URL


https://github.com/freshsong/enrolmentproject

🚩 Summary


참여자 : 최송이, 안효근, 정소진, 김재영, 류제욱 5인

담당 영역 : 최송이(수강신청하기, 수강신청미리보기, 작업초안 피그마작업), 안효근(전공자, 백엔드총괄), 정소진(프론트총괄), 김재영(출석체크 및 출석확인), 류제욱(관심강좌등록, 관심강좌 미리보기, 설문조사)

제작기간 : 2주

기여도 : 50%

그동안 공부했던 기술을 바탕으로 DB를 받아 적용할 수 있는 진짜 홈페이지 만들어보기.

학생 : 수강신청 / 내가 신청하나 강의 보기 / 즐겨찾기 / 만족도 설문조사 / 교수영역 접속불가

교수 : 학생 출석체크 및 출석부 확인 / 내강의 보기 / 학생영역 접속불가

🧐 주요기능

  1. 회원가입 / 로그인 / 로그아웃
  2. 수강신청하기, 클릭한 강의 시간표 미리보기
  3. 관심강좌 등록하기, 관심강좌 삭제하기
  4. 설문조사하기
  5. 팝업창 12시간 뒤 재활성화
  6. 출석체크, 출석부보기, 내수업보기

🔍 Meaning


프로젝트를 진행하기전에 걱정이 많았습니다. 말로만듣고 광연 이런 홈페이지 하나를 만들 수 있을까..? 라는 생각이 들었습니다. 우선 막막했지만 전공자였던 안효근님의 총괄 및 지시, 그리고 깔끔한 업무 분담으로 차근차근 시작할 수 있었습니다. 기본적인 기술만 배우고도 코딩을 하나 둘 입혀가며 실제로 화면에 찍히게 하는게 신기했고, 같은 결과물을 내더라도 방법이 다 다른다는 것을 또다르게 배웠습니다.

제일 어려웠던 점은, 전체적인 틀을 먼저 충분한 회의를 한 끝에 업무 지시서가 일별, 시간별로 나뉘어져 있었다면 개인 공부를 하면서 그날그날 할일을 조금씩 해놓을 수 있었을 것 같았습니다. 아쉬운대로 저는 제가 다룬지 얼마 되지 않은 피그마로 대략적인 자료들을 모아 공유를 했고, 그 공유한 부분을 조금씩 수정해가면서 각자 맡은 부분을 착실하게 해낼 수 있었습니다.

파워 J형으로써, 전체적인 틀을 싹 잡아 글꼴, 전체적인 구역별 색깔, 포인트 컬러등 세부적인 프론트 UI 부분들과 어떤 기능을 구사할 지 구체적으로 잡아야 업무 파악이 보다더 빨랐을텐데 모두의 스케줄을 조정하기 어려워 프론트는 프론트, 백엔드는 백엔드 살짝 속도가 맞지 않아 그런 부분에서 아쉬운 점이 있었습니다.

백엔드 맡은 부분을 구현했을 때, 머릿속에서 그려진 내용대로 실제로 코드를 짜서 구현하려 하니 잘 되지 않아 많잉 답답한 부분도 있었는데, 많잉 물어보고 찾아본 덕에 비교적 빠른 시간내에 기능을 구현할 수 있었습니다.

지금은 이 인원으로 SNS (인스타그램같은) 웹사이트 구상하려고 준비중이고, 팀원들 몯두의 힘이 합쳐진 결과 전체적으로 조화롭고 만족스러운 프로젝트 였습니다. 팀원 개개인의 구역이 확실히 나눠져있어서 업무파악이 비교적 수월했고 개인이 하지 않으면 바로바로 티가 났기 때문에 소홀히 할 수 없었던 프로젝트였습니다. 정말 재미있었고 또 시작하게 되는 프로젝트도 정말 기대중입니다.
