자바스크립트로 게임을 할 수 있다고?

🖇 Development URL


https://github.com/freshsong/parkingcar_js

🚩 Development URL


웹사이트 개발을 시작하며 HTML, CSS를 공부하고 있었습니다. 문득, 지금처럼 모바일 게임, 프로그램 게임등 고도화된 기술적인 프로그램으로 하는 게임이 아닌 지금 공부하고있는 HTML, CSS로도 간단한 게임을 구현하면 얼마나 좋을까? 생각했습니다.

HTML, CSS를 공부하는데 더 도움이 됐던것이 의미가 있었지만, 공부를 하면서 Javascript의 또다른 기능을 구현할 수 있었다는 점에서도 더 의미가 있었습니다.

HTML, CSS, Javascript 이외의 언어로도 다른 게임을 구현할 기회가 생기면 꼭 해봐야겠다 생각했습니다.


🧐 주요기능

  1. Javascript 자동차 그림 위치값을 알고 움직이기
  2. 주차성공시 "주차성공!" alert창 뜨게하기

🔍 Meaning

처음 개발을 공부했고, HTML CSS 모든것이 낯설었습니다. 차츰차츰 HTML, CSS를 익혀갈 쯤 JavaScript도 알았고 없어서는 안될 존재라는것도 알았습니다. 직접 사용해보고 느낀 것은, 풀스텍 언어를 배우고있지만 백엔드를 배우기 이전에 프론트의 역할도 만만치 않구나 느꼈습니다. 백엔드를 하나도 쓰지 않고서도 이런것들을 구현할 수도 있다는 신기한 감정을 느꼈고, 처음 Javascript를 접해서 해봐지만 처음이라 낯설고 어렵고 신기했습니다.

사진 2개로 간단한 게임도 구현하고 자바스크립트를 좀 더 열심히 파봐야겠다 생각했습니다.

마지막으로, 공부를 계속 하게되면서 자바스크립트 외의 언어가 어떤 기능이 숨겨져있을까 너무 기대됩니다.


🔨 Technology Stack
