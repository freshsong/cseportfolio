스프링으로 개인블로그 만들기

🖇 Development URL


https://github.com/freshsong/reshop

🚩 Summary


JSP로 게시판을 구현한 뒤 그 다음 단계인 Spring 으로도 비슷한 방법으로 블로그도 만들 수 있다고 하여 접목시키고 싶었습니다.

Mybatis를 이용해서 어노테이션과 Bean에 기능들을 담아 꺼내쓰는 방법들을 익혔습니다.

유튜브 링크를 따와서 직접 페이지에 넣는 작업도 가능하다는 방법도 알아냈고 앞으로 나만의 페이지도 만들 수 있다는 생각에 재미있었습니다.

Mybatis를 연동시키면서 페이지 내에서 많은 오류사항들이 발견되었지만, 오류를 해결해내는 과정속에서 나도 해낼수 있다는 쾌감을 맛보았습니다.


🧐 주요기능

  1. 회원가입 / 로그인 / 로그아웃
  2. 글등록, 댓글등록
  3. 내가 올린 관심사진, 관심유튜브 자료 확인

🔍 Meaning


처음 자바를 접하면서 JSP라는 방식으로 프론트엔드 - 백엔드 기능을 함께 접목시켜 하나의 페이지를 구사할 수 있다는 방법을 듣고 좀 신기하네? 싶었습니다. 이 작은 프로그램 하나로 무궁무진한 시각적 페이지를 구사하고 비시각적 데이터를 수집해서 하나의 결정체가 되는구나! 를 느낄 수 있었는데, Spring을 공부하고 구현하는 도중에 색다른 방법으로도 같은 기능을 구현시킬 수 있다는 것에 정말 재미있었습니다.

노트북으로 할때, 컴퓨터로 할때, 각각 작업환경이 달라짐에 따라서도 오류사항이 많이 발견되어서 처음에는 버거웠지만 서서히 오류사항을 하나씩 해결해 나가면서 와 이런 재미도 있구나..! 해결해내는 과정 속에서 짜릿한 쾌감이 있었습니다.

네이버블로그, 티스토리, 노션을 사용해오면서 내가 이런것들을 쓰면서도 만들수 있을까.. 항상 고민했었는데 이런 방법으로도 구현이 된다는것에 더 큰 쾌감이 있었습니다. 평소에도 잘 사용할 수 있을 것 같았고, 실제로 배포하게 되면 데이터 구축만 잘 해두면 다른 사람들도 밴드나 카카오톡처럼 이용 가능할 것 같다는 생각도 들었습니다.


🔨 Technology Stack

  1. Frontend : Javscript, CSS, HTML, Bootstrap
  2. Backend : Java(version11, Mybatis, Spring), MySQL
