골프장 웹사이트 만들기

🖇 Development URL


https://github.com/freshsong/sparkzone
https://sesparkzone.netlify.app/

🚩 Summary


개인프로젝트, 팀프로젝트 구상 중 처음에는 카페 사이트를 만들고 싶었는데 만들다보니 관심있는걸 하고싶다는 생각에 골프장 사이트 만드는 것에 관심을 가졌습니다.

혼자 프로젝트를 만들면서 예약, 게시판, 사진올리기, 마우스 우클릭금지, 클릭시 카드 화면전환.. 기능들을 구현하고 싶었습니다.

현재 프론트 작업 마친 뒤 백엔드 작업중인데, 생각보다 어렵지만 하나씩 해결되고 보여지는 결과물에 뿌듯해하며 진행중입니다.

🧐 주요기능

  1. 회원가입 / 로그인 / 로그아웃
  2. 예약하기
  3. 문의등록 / 답변달기 / 갤러리 모음

🔍 Meaning


프로젝트를 작게작게 진행을하면서 메인 홈페이지만 만드는것이 아닌, 정상적으로 구현할 수 있는(추후에 쓸수도 있는) 작업물을 제 손으로 100% 작업하고 싶었습니다. 유튜브를 보고있는데 박세리 골프 장면이 나왔고, 처음에는 카페 홈페이지 메인을 만들면서 갸우뚱 하면서 박세리 골퍼를 보는순간 이거다! 싶어서 앉은자리에서 바로 작업을 진행했습니다.

현재 프론트 작업은 일부 마친 뒤 에러를 잡고 자바 작업중입니다. 회원가입 로그인까지 구현했는데, 하나씩 구현해내며 해낼 수 있다는 자신감에 코딩이 정말 흥미롤웠고 또다른 작업물도 만들 수 있다는 긍정적인 생각까지도 생겼습니다.

현재 작업을 진행중이지만, 구현을 100% 완성시켜내서 추후 만뜰려고 했던 컼피 사이트도 한번 근사하게 만들고 싶었습니다.


🔨 Technology Stack

  1. Frontend : Javscript, CSS, HTML, Bootstrap
  2. Backend : Java(version11), MySQL
