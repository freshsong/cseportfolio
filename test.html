메이플 경매장 알리미
2023.08 - 2023.09  (2人 팀 프로젝트)
2/3
메이플스토리의 아이템이 원하는 조건으로 경매장에 올라왔을 때 알림을 받아보기 위한 웹사이트 입니다. 메이플스토리를 좋아하는 한 친구의 아이디어에 착안하여 개발하였습니다.

새롭게 출시된 React 18 과 Next.js 13을 적극적으로 공부하고 활용해볼 수 있었을 뿐 아니라, React-Query와 React-Hook-Form 등의 편의를 돕는 여러 라이브러리도 활용해볼 수 있었습니다.

Tailwind CSS 에 조금 더 익숙해지는 계기가 되었고, Tailwind CSS 기반으로 다크 모드 를 구현해보고 Flowbite 라이브러리를 통해 각종 컴포넌트를 가져와 사용해볼 수 있었습니다.
