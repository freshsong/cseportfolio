LH 홈페이지 따라해보기

🖇 Development URL


https://github.com/freshsong/lhproject

🚩 Summary


개발 공부에 어느정도 감이 익혀질쯤, 코딩을 많이 해봐야 빨리 실력 향상이 될거다~ 라는 생각에 간단한 메인페이지 하나를 만들고 싶었습니다.

HTML, CSS 외에 부트스트랩이라는 기능도 넣을 수 있다는 말을 듣고 여기저기 찾아보고 최대한 구현하고 싶었습니다.

공부한 것들을 까먹지 않기 위해서 header / footer / main 각각 나눠서 화면에 구현시키고 싶었고 공부하면서 익혔던 내용도 빠짐없이 넣고 싶었습니다.


🧐 주요기능

  1. Header nav 메뉴 그대로 구현
  2. 오른쪽 상단 메뉴박스 내의 구역 그대로 구현
  3. scroll 박스 짤리지 않게 구현

🔍 Meaning

개발 공부를 하면서 홈페이지는 당연히 누군가의 프로그램으로 만들겠지 생각했는데, 직접 구현할 수 있다는 것 그리고 그걸 제가 해낼 수 있다는 것에 굉장히 큰 매력을 느꼈습니다.

단순히 종이를보고 체크하고 입력하는 것들이 아닌, 내 머릿속 생각을 그때그때 구현시키며 바로바로 확인 가능하고 피드백 할 수 있다는 점에서 놀라웠습니다.

최대한 비슷하게 따라하려고 많이 비교해보고 찾아보고 코딩했는데, 만족스럽습니다.

웹페이지 코딩에 조금 자신감이 붙었고, 구조에 대해서 보다더 확실하게 알 수 있었습니다.


🔨 Technology Stack

  1. Frontend : Javscript, CSS, HTML, Bootstrap
